Descripción detallada

Modela un filtro de suavizado (pasa bajos), encargada de filtrar una imagen.

Aplica un filtro llamado filtro de suavizado (pasa bajos), el cual toma valores de pixeles vecinos para modificar la intensidad del pixel actual mediante la aplicacion de un kernel correspondiente.
