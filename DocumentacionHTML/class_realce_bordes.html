Descripción detallada

Modela un filtro de realce de bordes (pasa altos), encargada de filtrar una imagen.

Aplica un filtro llamado filtro de realce de bordes (pasa altos), el cual toma valores de pixeles vecinos para modificar la intensidad del pixel actual mediante la aplicacion de un kernel correspondiente.
